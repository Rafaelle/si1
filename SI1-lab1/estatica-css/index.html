Disciplina Sistemas de Informação 1 - SI1

Informações Gerais: O objetivo da disciplina é discutir técnicas de projeto de software. Haverá foco particular em técnicas utilizadas para melhorar o reuso e a flexibilidade de software desenvolvido usando Orientação a Objeto, mas discutiremos também projeto arquitetural. O contexto de todas as aplicações será a web. Assim, ao término da disciplina você também saberá desenvolver sistemas web.

Professor: Nazareno Andrade

Avaliação: A nota é composta por 3 tipos de avaliação: miniteste, labs e um projeto.

Disciplina Pré-requisito: Gerência da informação.

Pré-requisito para: Sistemas de Informação II e Banco de Dados I.
